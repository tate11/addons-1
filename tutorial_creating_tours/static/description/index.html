This app is a tutorial in the form of code. The app will learn you how to create tours in Odoo.

Learn how to create tours

This app contains all the code you need to add a tour to Odoo and how to show and translate tour steps.
All the code is documented inline with notes and extra information to understand how tours work.

Contains

  • New tour named "example_tour".
  • Example of translations for tours (Dutch).
