This app is a tutorial in the form of code. The app will learn you how to inherit existing QWeb reports in Odoo.

Learn how to inherit QWeb reports

This app contains all the code you need to inherit existing QWeb reports in Odoo.
All the code is documented inline with notes and extra information to understand how inheriting QWeb reports works.

Contains

  • New QWeb record that inherits the existing quotation/order report.
  • Adds the phone and email on the quotation and a warning message on the quotation/order.
  • Inline comments explaining how to inherit an existing report.
  • PO file showing how to translate modified QWeb reports.
