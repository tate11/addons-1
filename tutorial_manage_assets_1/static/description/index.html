This app is a tutorial in the form of code. The app will learn you how to create, add and use JavaScript files in Odoo.

Learn how to add JavaScript to your modules

This app contains all the code you need to add JS files into an Odoo module.
All the code is documented inline with notes and extra information to understand how the feature works.

Contains

  • New JavaScript file 'hello_world.js' that logs a console message.
  • New XML file that adds the JavaScript to the Odoo assets bundle.
  • Inline code explaining how the functionality works.
