This app is a tutorial in the form of code. The app will learn you how to modify existing pivot and graph views in Odoo.

Learn how to modify existing pivot/graph views

This app contains all the code you need to modify existing pivot and graph views in Odoo and how to add custom measures.
All the code is documented inline with notes and extra information to understand how inheriting existing graphs works.

Contains

  • Inherits the existing pivot and graph from the sales analysis (Sales > Reporting > Sales).
  • Adds custom fields on the models "sale.order" and "sale.order.line" and adds them to the report.
  • Creates new measures on the sales analysis (Kickback fee, Monthly profit, One time profit and Probability).
