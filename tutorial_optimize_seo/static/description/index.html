This app is a tutorial in the form of code. The app will learn you how to add/optimize SEO in Odoo and how to create customized social media previews.

Learn how to add/optimize SEO and social media previews

This app contains all the code you need to add and optimize SEO for your own models.
Learn how to customize social media previews for your records/models.
All the code is documented inline with notes and extra information to understand how the SEO features work.

Contains

  • New app on homescreen named "Tutorials".
  • New menuitem named "Tutorial".
  • New model named "tutorial" in the database.
  • New form, tree and search view under the menuitem "Tutorial".
  • New controller for tutorials
  • New web url's http://yourwebsite.com/tutorials and https://yourwebsite.com/tutorial/id showing the main information. You can see the overview and access the tutorials from http://yourwebsite.com/tutorials
  • Override of the function '_default_website_meta' to create customized social media previews and SEO
  • Example data filling up the tutorial view on demo databases.
