This app is a tutorial in the form of code. The app will learn you how to create chatter functionalities and activities in Odoo.

Learn how to add an archiving option to your models

This app contains all the code you need to add the ability to archive records.
All the code is documented inline with notes and extra information to understand how the archiving feature works.

Contains

  • New menuitem named "Contact categories" under the "Contacts" app.
  • New model named "contact.category" in the database.
  • New form and tree view under the menuitem "Contact categories".
  • New field 'Category' on the contact model.
