This app is a tutorial in the form of code. The app will learn you how to create security groups in Odoo.

Learn how to create security groups

This app contains all the code you need to add a security group to Odoo.
All the code is documented inline with notes and extra information to understand how security groups work.

Contains

  • New app on homescreen named "Access rights demo".
  • New menuitem named "Rights demo".
  • New security groups named "User" and "Manager" under the access rules.
  • New security option on the users profile named "Demo module access".
