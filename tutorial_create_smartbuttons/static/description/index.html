This app is a tutorial in the form of code. The app will learn you how to create new smartbuttons in Odoo.

Learn how to create new smartbuttons

This app contains all the code you need to add a new smartbutton in Odoo.
All the code is documented inline with notes and extra information to understand how smartbuttons work.

Contains

  • New smartbutton under contacts > configuration > countries showing how many contacts are from a country.
  • Clickable smartbutton that opens all contacts for a specific country.
  • Action to open up filtered contacts (in Python).
